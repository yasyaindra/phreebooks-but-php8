PayPal nvp Payment Module

Developed by PhreeSoft, LLC.
Created 2011-07-01 from PhreeBooks version 2.1 to new Phreedom framework

For Preedom ERP Payment Method

Install
1. Copy paypal_nvp folder to the /modules/payment/methods folder
2. Navigate to Company -> Module Administration -> Payment Module Properties Icon
3. Press Install next to the PayPal Payment Pro method (must have full permissions to install)
4. Press the Properties icon next to the PayPal method and enter required fields.

Removal
1. Navigate to Company -> Module Administration -> Payment Module Properties Icon
2. Press Remove next to the PayPal Payment Pro method (must have full permissions to remove)
3. Optional - remove folder /modules/payment/methods/paypal_nvp
