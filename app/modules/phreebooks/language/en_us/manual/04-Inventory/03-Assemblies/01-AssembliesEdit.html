Inventory - Assemblies (Edit)

Edit inventory adjustment procedure

  1. From the main menu, select Inventory->Assemblies.
  2. Press the Open button to display the inventory assemblies popup window.
  3. Select the assembly to edit by clicking anywhere on the row of the assembly.
  4. Change any fields as necessary.
  5. Press Save to post the transaction to the journal. Note: Editing a journal entry first un-posts the original transaction and re-posts with the new data. Cost of good sold will be recalculated based on the post date of the edited adjustment.
