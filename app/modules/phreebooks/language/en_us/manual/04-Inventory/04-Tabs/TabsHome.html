Inventory - Tabs (Categories)

PhreeBooks has a built in feature to expand inventory attributes (fields) to allow for further refinement of your inventory items. To help organize these expanded fields, added inventory fields are organized into categories. These categories become the tab headings when maintaining your inventory items. The mandatory inventory fields are, by default, put into the 'System' tab. If the inventory type is an assembly, a second tab name 'Bill of Materials' is also added. At least one tab must be added before inventory fields can be entered.

Adding an Inventory Tab (Category)

  1. From the main menu select Inventory->Setup Inventory Field Tabs.
  2. Press the New button.
  3. Enter a short name for the inventory tab, 10 characters or less, no spaces.
  4. Enter the sort position. This will determine the display order of the tabs when maintaining inventory items.
  5. Enter a tab description. This field will show up below the tab to describe the contents of the tab.
  6. Press Insert to add the new tab.
