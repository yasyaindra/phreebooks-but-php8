Importing and Exporting Data

PhreeBooks has a powerful import/export tool to transfer data in to/out of the PhreeBooks database. Knowledge of the PhreeBooks database structure and data fields are necessary to acurately import information into the database. Exporting is less complicated but still required some knowledge of the database tables.

Most import/export operations deal solely with a single database table. The exception is the account definitions (customers, vendors, branches). These definitions include the address book fields (currently set at 5) with the main account information.

Other import scripts specialized towards beginning balance initialization can be found here.

The HOW TOs contain samples of import procedures to import customers, vendors, and inventory.
