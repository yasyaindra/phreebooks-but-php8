Setup - Users

PhreeBooks user accounts allow for individualized access to the PhreBooks modules. User accounts are used in the audit log, setting access permissions, and permissions for reports.

When the PhreeBooks systems is installed, a single administrator account is added given full permission to access all of the PhreeBooks functionality. New users can be added and thier access can be controlled to modules that they are given permission to access. If No Access is given to a function, the option will not appear in their menu when they log in.

NOTE: When adding new modules, permissions have not been established and the modules will not be visible. Each user account will need to be edited and the appropriate permission added to access the module.

Adding a User Account

  1. Select Setup->Users.
  2. Press the Add icon. A new screen will appear with a form to fill out for the new user.
  3. Enter a username, email address, and password for the new user.
  4. Set the permissions for the user. A tab will appear for each main heading. the permissions are initially set for no access for all modules. Select the permission levels for each module in each tab.
  5. Press the Save icon to create the user account.

Editing a User Account

  1. Select Setup->Users.
  2. Press the Edit icon next to the user to edit. A new screen will appear with a form fill out for the new user.
  3. Edit any user information and security permissions as necesary. If the password will no be changed, leave the password fields blank and no changes will be made to the current password in the database.
  4. Press the Save icon to update the user account.

Deleting a User Account

  1. Select Setup->Users.
  2. Press the Delete icon next to the user to delete. A popup will appear to confirm the selection.
  3. Press OK to confirm the selection and the user account will be removed from the database.
