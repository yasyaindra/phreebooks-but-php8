Currencies Setup

PhreeBooks supports multiple currencies. The default currency is set during setup but additional currencies can be entered through the Setup->Currencies menu.

To enter a new currency:

  1. Select Setup->Currencies.
  2. Enter the fields request in the popup form.
  3. Press the Save button to save the record.

To delete a currency:

  1. Select Setup->Currencies.
  2. The current currencies are listed. To delete a currency, select the delete icon for the currency to delete.
  3. Confirm the deletion of the record. NOTE: If the default currency is deleted, a new default currency should be selected.
