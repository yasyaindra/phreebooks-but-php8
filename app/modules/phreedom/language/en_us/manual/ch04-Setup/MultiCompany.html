Setting Up Multiple Companies

PhreeBooks requires an unique database for each company.

NOTE: Prior to adding or copying a new company, the database must exist in the system. Phreebooks WILL NOT generate the database!

Adding a new company

  1. Go to Tools->Company Manager.
  2. Enter the database name in the Add New Company fieldset.
  3. Enter the company full name in the Add New Company fieldset.
  4. If you want the demo data loaded, check the Add Demo Data box.
  5. Click the Add New Company button.

Note: When a new comany is created, a directory under /my_files is created named the same as the database. Additional directories and system files are created under this new directory to store uploads, logs and other information pertaining to the newly created company.

Copying a company

  1. Go to Tools->Company Manager.
  2. Enter the database name in the Copy Company fieldset.
  3. Enter the company full name in the Copy Company fieldset.
  4. Check the boxes for the information you want to transfer over to the new company.
  5. Click the Copy Company button.

Note: When a company is copied, a directory under /my_files is created named the same as the database. Additional directories and system files are created under this new directory to store uploads, logs and other information pertaining to the newly created company.

Deleting a company

WARNING: THIS IS A NON-RECOVERABLE EVENT! ALL DATA WILL BE LOST! BE SURE TO BACK UP YOU FILES BEFORE DELETING A COMPANY.

  1. Select the company from the pulldown menu in the Delete Company fieldset.
  2. Click the Delete Company button and confirm the deletion.
  3. All company data will be erased. This includes all database tables as well as the directory (and subdirectories) under the /my_files/ directory named after the database.
