Shipping Modules - United States Postal Service (USPS)

PhreeBooks includes a shipping module to calculate United States Postal Service (USPS) shipping charges.

Features Include:

  • Freight quotes
  • Shipping Manager integration
  • Customer and Vendor order integration.
  • Integration with Invoice Manger
  • Tracking information and shipping logs
  • Report capability available through reportwriter.

How to Install USPS Shipping Module

  1. Through Setup -> Shipping Settings, click on the USPS row in the table and a button labeled Install will appear on the right.
  2. Click on Install to install the USPS module.
  3. Click the USPS row again and Click on the Edit Button.
  4. Enter the rate information and press Save to save the changes.

Note: Place the sort order to a number greater than zero. IF THE SORT ORDER IS ZERO, IT WILL NOT SHOW UP.

How the Get a Rate Quote

  1. Rate quotes are available from the order screens. When shipping is enabled (Company -> My Company -> Enable Shipping Settings) a button will appear next to the freight method and price to load the shipping popup and request a rate.
  2. The Shipping Quote popup will be mostly filled out with values pulled from the order and the shipping defaults. Changes , however can be made before the carrier servers are queried for rates.
  3. Once the information in the shipping popup has been verified, Press the Next button and the rate estimator will pull the quoted rates form the selected services.
  4. Click on the rate desired and the carrier, method, and rate information will be entered into the order form; the popup will close.

Note: If the weight field of inventory items is filled out, the weight of the shipment will be automatically calculated and inserted into the rate estimator.

How to Enter a Shipping Log Record

  1. Shipping log entries are entered either through the Invoice Manager.
  2. From Customers -> Invoice Manger, locate the invoice to create the shipping log entry.
  3. Press the Ship button on the line the invoice appears.
  4. A popup will appear with a form to enter the package information.
  5. Select the Service Type, shipment/Expected receipt dates, tracking information and the shipment cost.
  6. Press the Save icon to save the shipment information. NOTE: Once a shipment log entry has been generated, the button on Invoice Manager changes from Ship to Void Shipment.

How to Void a Shipping Log Record

  1. From Customers -> Invoice Manger, locate the invoice you want to void the shipping log entry.
  2. Press the Void Shipment button and the shipping log entry will be deleted. NOTE: The Invoice Manager will now allow the package to be re-shipped if necessary.
  3. From Tools -> Shipping Manager, locate the shipment to void.
  4. Press the Delete icon on the line of the shipment, confirm the void and the shipment record will be deleted.

How to Track a Package

  1. From Tools -> Shipping Manager, select the date the package was shipped. Locate the package to track.
  2. Since USPS shipping is not tied to a particular vendor, only the tracking number will be displayed, it can now be copied to the clipboard and pasted into the carrier site to track the package.
