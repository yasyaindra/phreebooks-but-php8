SOAP/XML Receivables Interface

PhreeBooks has an SOAP/XML interface build in to support XML type transfers that allow data to be inserted into and generated from the system. This will allow interfaces to third party applications and data transfers. Current XML definitions exist for:

Sales Orders

Submit Sales Orders (Import)

Sales Invoices

 

Future Definitions

Generate Sales Order
Cancel Sales Order
Status Sales Order
Invoice Sales Order
Update Sales Order

Submit Sales Invoice
Update Sales Invoice
Ship Sales Invoice
