Sales Order Manager

The sales order manager is a powerful tool to manage your companies sales orders. From this function, you can enter and review expected shipment dates, print copies of the sales order, or edit the sales order. The sales order manager default to the current period but a search field and period selection pull down are provided to help locate SOs anywhere in the system.

Sales Order Manager Features:

  • Easily view sales orders and determine open/closed status and expected shipment dates
  • Powerful search capabilities to find or filter results
  • Ability to print SOs without opening them up or re-posting
  • Ability to set expected shipment dates so you know when shipments are expected, helps manage inventory
  • Ability to open for edit the sales order and review additional details
  • Ability to invoice a sales order

Viewing a Sales Order

  1. Select Customers->Sales Order Manager
  2. Locate the sales order to view and press the icon labeled Edit.
  3. The Sales Order screen will load and the form will be filled in with the information of the selected SO.
  4. From here, the sales order may be edited, deleted, closed or reviewed.

Printing a Sales Order

  1. Select Customers->Sales Order Manager
  2. Locate the sales order to print and press the icon labeled Print.
  3. The reportwriter form display screen will appear listing the available forms to print.
  4. Select the form to print, the delivery method, and press the PDF icon to generate the form.

Invoicing a Sales Order

  1. Select Customers->Sales Order Manager
  2. Locate the sales order to invoice and press the button labeled Invoice.
  3. The Sales/Invoice screen will load and the remaining items from the selected SO will be shown.
  4. Enter in the quantities to ship for each line item.
  5. Make any other changes as necessary; such as the customers purchase order number shipment details, etc.
  6. Press the Save icon to save the invoice or the receipt icon to receive payment form the customer.

Changing Delivery Dates for a Sales Order

When a sales order is placed, the expected shipment date is set to the post date. The estimated shipment dates may be modified here to reflect the current estimated shipment dates for each line item on the SO. the shipment dates field is useful to identify when items are expected for filling back-orders and notifications of expected arrivals of inventory items.

  1. Select Customers->Sales Order Manager
  2. Locate the sales order to adjust or view the expected shipment dates and press the button labeled 'Delivery Dates'.
  3. A popup will appear listing each line item from the SO, the quantity ordered and the description. The currently expected shipment date will also be displayed.
  4. If necessary, enter a new shipment date for each item on the SO. A popup calendar is provided to help locate dates. If no change is necessary, leave the date field blank to retain the current shipment date.
  5. After updating all affected shipment dates, press the Save icon the save the charges and return to the Sales Order Manager window.
