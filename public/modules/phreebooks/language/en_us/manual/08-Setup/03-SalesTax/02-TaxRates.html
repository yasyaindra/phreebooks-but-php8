Setup - Sales Tax Rates

Sales tax rates can be set at several levels depending on the taxing structure of your business. Prior to setting up sales tax rates, sales tax authorities need to be set up. Refer the the help file on Sales Tax Authorities for more information.

Adding a Sales Tax Rate

  1. Select Setup->Sales Tax Rates.
  2. Press the New icon. A popup will appear with a form to fill out.
  3. Enter a short name for the tax rate, e.g. Local Tax, State TAX, VAT, etc.
  4. Enter a more detailed description of the tax rate.
  5. To add or delete a taxing authority, select it from the appropriate list. NOTE: Only one authority can be added or removed at this time. To add more than one authority, save the entry and then edit it to add more.
  6. If freight is taxable, selct the Yes radio box.
  7. Press the Save icon to commit the entry to the database.

Editing a Sales Tax Rate

  1. Select Setup->Sales Tax Rates.
  2. Press the Edit icon next to the sales tax rate to edit. A popup will appear with a form filled out with the current tax rate information.
  3. Edit the short name or description as necessary.
  4. To add or delete a taxing authority, select it from the appropriate list. NOTE: Only one authority can be added or removed at this time. To add more than one authority, save the entry and then edit it to add or remove another authority.
  5. If freight is taxable, selct the Yes radio box.
  6. Press the Save icon to commit the change to the database.

Deleting a Sales Tax Rate

  1. Select Setup->Sales Tax Rates.
  2. Press the Delete icon next to the sales tax rate to delete. A confirmation popup will appear to verfiy the delete request.
  3. Press OK to remove the sales tax rate from the database.
