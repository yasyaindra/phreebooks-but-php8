Setting up Multiple Locations (Branches)

PhreeBooks supports multiple branches/stores. The company information is gathered during setup but branch information is entered through the Setup->Maintain Branches menu.

To enter a new branch:

  1. Select Setup->Maintain Branches.
  2. The current branches are listed. To edit a branch, select the edit icon for the branch to edit, or, to enter a new branch press the Add button.
  3. Enter a branch ID. Branch ID's should not contain spaces or special characters. It is best to use an abbreviation or store number as the Branch ID.
  4. Enter the Branch information requested by the form. Required information is identifed by a * next to the input box.
  5. Press the Save button to save the record.

To delete a branch:

  1. Select Setup->Maintain Branches.
  2. The current branches are listed. To delete a branch, select the delete icon for the branch to delete.
  3. Confirm the deletion of the record. NOTE: If the branch is used in a journal record, it cannot be deleted. Instaed, check the inactive box and the branch will no be shown on the pull down menus for the order forms.
