Users and Security

Phreebooks support multiple users each assigned multi-level security. During setup, the administrator user is defined and given full privileges. the administrator can then add additional users and set security appropriately. Each module has an unique securty id and can be set to the following levels:

  • None - No access. Users with this security will not see the module as part of the menu.
  • Read Only - The user can only open and read the information. they are not permitted to save and changes they make to the record. In this case, the Save icon will not be displayed.
  • Add - The user can create records but cannot edit them or delete them.
  • Edit - The user can add and edit a record but cannot delete the record.
  • Full - The user has full read/write access to the record.

The add/edit a user:

  1. Select Setup->Users.
  2. The current users are listed. To edit a user, select the Edit icon for the user to edit, or, to enter a new user press the Add button.
  3. Enter/edit the fields as necessary. Enter the security levels for each module as necessary.
    NOTE: If editting a user and the passwords fields are left blank, the password will not be affected.
    NOTE: Users may be linked to the employee list only if the employee us set up prior to the user account.
  4. Press the Save icon to save the record. NOTE: the user settings will not take effect until the user logs off and back onto the system.

To delete a user:

  1. Select Setup->Users.
  2. The current users are listed. To delete a user, select the Delete icon for the user to delete.
  3. Confirm you choice and the user will be removed from the database.
