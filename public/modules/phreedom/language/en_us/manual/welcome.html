Welcome to PhreeBooks

PhreeBooks is an open source accounting application designed for ease of use and robust functionality. Written in freely available PHP and mysqli, PhreeBooks has been tested on multiple platforms and browsers to ensure the broadest compatibility possible.

Some of the feature highlights are:

  • Web based interface
  • Full featured accounting engine
  • Complete inventory control including assemblies
  • Multiple company capablility
  • Multiple language capablility
  • Robust reporting capability
  • Full featured import/export capability
  • User friendly installation and setup script
  • Extensive user level security

PhreeBooks is release under the GNU license.
